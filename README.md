## 🎮 About Me

Game Development & Programming graduate building games with Unity & Godot.
I love crafting systems from the ground up, experimenting with mechanics
and designing luck-driven game loops — with a growing focus on online & multiplayer experiences.

- 🔭 Currently building: **Caulamity** — a 3D physics-based FPS cooking chaos game
- 🎰 Latest release: **Slot Survivor** — a solo-built roguelite with slot machine upgrades
- 🏆 **3rd place** at Sandwich Jam II with **BoomBurn** (team of 4)
- 🌱 Exploring: multiplayer networking, WebGL optimization & game architecture patterns
